🚦 Red Lines enforced by the skill
No ghostwriting
The skill may draft bullets and outlines from the student's own evidence, but never produces a complete application essay the student didn't write.
FAFSA/FERPA citation required
Every regulated claim must be backed by rag.baselineContext.ruleCitations. No citation = "check your counselor."
Crisis passthrough
_meta.topicType === CRISIS → display backend crisis resources verbatim. The skill never attempts to counsel.
Korea PIPA consent gate
HTTP 403 with consentRequired: "cross_border_transfer" → stop and direct to consent flow. Never reroute providers.
🔑 BYOK — Per-Student API Key Flow
1

Student submits key via PUT /api/students/apikey

Request body: { apiKey, provider?, baseUrl?, defaultModels?: {small, medium, large} }
Age gate checked first — BYOK blocked for minors. Model ids validated with isReasonableModelId() before network is touched.

2

Auto-detect provider from key prefix

sk-ant-* → anthropic · sk-or-* → openrouter · sk-* → openai · AIza* → google
Explicit provider param always wins. Custom baseUrl can point at any OpenAI-compatible host.

3

Validate key via adapter ping

adapterValidateKey({ provider, apiKey, baseUrl }) sends a 1-token request to confirm the key is live. Auth-rejected (401/403) → HTTP 400 "API key rejected". Other errors (bad model id on compat host) → "probably valid".

4

Encrypt and store in pii-vault.db (AES-256-GCM)

Columns added: provider, base_url, default_small_model, default_medium_model, default_large_model
Pre-existing rows auto-migrated to provider = 'anthropic'.

5

Used in every subsequent /api/llm call for that student

lookupStudentBYOK(piiStmts, piiVault, studentId) decrypts the key per-request. Priority: request.apiKey → byok.apiKey → ENV.
Narrative-fit internal calls (callHaikuForNarrativeFit) also use BYOK via byokLookup callback — scoring runs on the student's bill.

✓

Response returns full context

{ stored: true, hint: "sk-ant-api0...c0f4", provider: "anthropic", baseUrl: null, defaults: {small, medium, large}, verified: true }
GET /api/students/apikey now also returns provider, baseUrl, and defaults.
